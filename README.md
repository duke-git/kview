# kview

> A UI components library based on vue.js.
> 前端组件库，基于vue2.0。目前版本v1.0.0。用于中后台管理系统的开发。

## Build Setup

``` bash
# 安装依赖 
npm install
要求：node >= 4.0.0, npm>= 3.0.0

# 开发环境localhost:8080
npm run dev

# 生产环境
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# 单元测试
npm run unit

# 所有测试
npm test
